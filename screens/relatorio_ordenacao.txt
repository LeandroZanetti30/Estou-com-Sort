Explicação do algoritmo:
Smooth Sort é um algoritmo de ordenação adaptativo desenvolvido por Edsger Dijkstra. É uma variação do Heap Sort que usa uma sequência de números de Leonardo para criar uma floresta de heaps, tornando-o mais eficiente para dados parcialmente ordenados.

Complexidade:
- Pior caso: O(n log n)
- Melhor caso: O(n) - quando os dados já estão ordenados
- Caso médio: O(n log n)

Características:
- Não é estável: pode alterar a ordem de elementos iguais
- In-situ: requer apenas uma quantidade constante de espaço adicional
- Adaptável: mais eficiente com dados parcialmente ordenados
- Baseado em números de Leonardo (L(0)=1, L(1)=1, L(n)=L(n-1)+L(n-2)+1)
- Complexidade de caso ideal O(n) quando a entrada está quase ordenada
